## Getting Started

### Prerequisites

- Install [Node.js](https://nodejs.org/).
- Refer to the `.mock.env` file and create the `.env` file in the root directory.

### Run

development mode

```bash
yarn install
yarn start:dev
```

production mode

```bash
yarn install
yarn build
yarn start:prod
```
